\subsection{Ransomware Detection Approaches}

\textbf{Signature-Based Methods:} Traditional antivirus systems rely on known 
malware signatures but fail against zero-day and polymorphic ransomware~\cite{ref1}.

\textbf{Behavior-Based Detection:} Systems monitoring file system operations, 
registry changes, and process behavior show promise but suffer from high false 
positive rates~\cite{ref2}.

\textbf{Machine Learning Approaches:} Recent work applies supervised learning 
(SVM, Random Forest, neural networks) to detect ransomware from behavioral 
features~\cite{ref3, ref4}. However, most lack interactive demonstration capabilities.

\textbf{Hybrid Systems:} Combining multiple detection strategies improves accuracy 
but increases complexity~\cite{ref5}.

\subsection{Demo and Interactive Systems}

Few existing systems provide interactive demonstration environments for exploring 
ransomware behavior and detection mechanisms in real-time~\cite{ref6}.
